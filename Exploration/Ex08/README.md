주석 또는 doc string을 보고 해당 코드가 잘 이해되었나요?**

- [ ]  **3. 에러가 난 부분을 디버깅하여 문제를 해결한 기록을 남겼거나
새로운 시도 또는 추가 실험을 수행해봤나요?**
    - 디버깅 과정에 대한 구체적인 기록은 부족한 편이었으나, 기존 개발 환경의 불안정성으로 Google Colab 환경으로 전환하여 문제를 해결하려는 노력이 돋보였음
    - 이를 통해 학습과 실험을 안정적으로 진행할 수 있었고, 결과적으로 모든 과제를 잘 수행함

# 회고(참고 링크 및 코드 개선)
어려운 개발환경에서도 방법을 찾아서 해결하신 부분에서 저도 갑작스러운 문제에 대응할 수 있도록 다른 개발환경도 같이 구축해야겠다는 생각이 들었습니다. 그리고 어려운 부분, 이해하지 못한 부분이 있었는데 수호님이 작성하신 코드와 회고를 참고하여 아직 끝내지 못한 제 과제에 참고할 수 있을 것 같습니다.
